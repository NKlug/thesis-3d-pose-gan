		\begin{itemize}

			\item Using only 2D poses -- no ground truth required
		\end{itemize}

			\item Very good 2D keypoint detectors out there: Stacked Hourglass, OpenPose etc.
			\item Getting 3D ground truth data is hard and expensive
		\end{itemize}
